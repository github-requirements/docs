C1 - Criando uma organização

| Titulo | Criando uma organização | | ————- | ————- | | Objetivo | Criar uma organização do GitHub. | | Contexto | Pré-condição: Ter uma conta do GitHub. | | Atores | Usuário. | | Recursos | Nome da organização e e-mail. | | Episódios | Usuário abre a aba de seu perfil e entra em configurações. | | | Usuário acessa a parte de organizações e clica em “New organization”. | | | Usuário adiciona o nome e o e-mail da organização. | | | Usuário escolhe o plano de pagamento. | | | Usuário aceita os termos de serviço e política de privacidade. | | | Usuário adiciona o nome da empresa ao qual a organização pertence. | | | Usuário confirma a criação da organização. | | | Usuário torna-se Colaborador da organização. | | Restrição | Ter o link para as configurações de seu perfil. | | Exceção | Nome da organização inválido. |

C2 - Convidando membros para a organização

| Titulo | Convidando membros para a organização | | ————- | ————- | | Objetivo | Convidar usuários para se tornarem membros de uma organização. | | Contexto | Pré-condição: Ter status de colaborador em uma organização. | | Atores | Colaborador e Sistema. | | Recursos | Nome do usuário ou e-mail. | | Episódios | Usuário clica em sua foto de perfil e em “Your Profile”. | | | Colaborador seleciona sua organização. | | | Colaborador seleciona a aba “People” e clica em “Invite member”. | | | Colaborador insere o nome ou o e-mail do usuário que se deseja convidar. | | | Colaborador escolhe os privilégios do usuário na organização. | | | Colaborador confirma o envio do convite. | | | Sistema envia um e-mail com o convite para o usuário. | | Restrição | Ter o link da página principal de sua organização. | | Exceção | Usuário não encontrado. |
